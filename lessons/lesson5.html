Hyper-V and Shared Folders

Install Hyper-V and Shared Folders

  1. Login to your Virtual Machine.
  2. Right-click the Windows Start button and select Windows PowerShell (Admin).
  3. Select Yes on the User Account Control window.
    Note: Running PowerShell as an administrator ensures you have the necessary permissions to install features.
  4. Run the following command:
    Enable-WindowsOptionalFeature -Online -FeatureName Microsoft-Hyper-V -All
  5. Enter Y when prompted to restart the computer to complete the installation.

Part III. Configure a New Virtual Machine

  1. Open Hyper-V Manager by typing Hyper-V Manager in the search field on the taskbar and selecting it.
  2. Select BSIT2<Section>-<YourLastname> in the left pane of the Hyper-V Manager window.
  3. In the Actions menu on the right, select New and then click Virtual Machine.
  4. Click Next on the Before You Begin page of the New Virtual Machine Wizard.
  5. On the Specify Name and Location page, enter KaliVM-<yourlastname> in the Name field, then click Next.
  6. Select Generation 1 on the Specify Generation page and click Next.
  7. On the Assign Memory page, enter 1024 in the Startup Memory field, then click Next.
    Note: Always allocate resources carefully to avoid over-provisioning.
  8. Click Next on the Configuring Networking page.
  9. On the Connect Virtual Hard Disk page, enter 90 for the Size, then click Next.
  10. On the Installation Options page, select Install an operating system later.
  11. Click Finish to complete the virtual machine setup.
